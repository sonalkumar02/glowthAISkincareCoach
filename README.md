# Glowth AI Skincare Coach

Glowth AI is a skincare assistant that helps users understand their skin, build a simple routine, and stay consistent with healthy skincare habits.

## What It Does

- Guides users through a skincare onboarding flow
- Helps users scan and track their skin routine progress
- Shows personalized skincare recommendations
- Tracks streaks and weekly progress
- Supports reminders so users can stay consistent
- Includes dashboard pages for routine, progress, profile, and recommendations

## Why Users May Like It

Glowth is built for people who want skincare guidance without confusion. Instead of guessing products or routines, users can follow a more organized skincare journey with reminders, progress tracking, and simple recommendations.

## Current Status

This project includes the public website and dashboard experience for Glowth AI. The app is being prepared for production use and future mobile app deployment.
